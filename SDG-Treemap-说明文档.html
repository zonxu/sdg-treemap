腾讯 SDG Treemap 可视化项目 说明文档 · 2024.04

本项目以 Treemap（树图）形式，将腾讯在联合国 17 个可持续发展目标下的实践项目进行数据可视化呈现。支持 2024 / 2023 两个年份、中文 / 英文两种语言，共 4 个独立页面。每个页面为纯前端单文件（HTML + CSS + JS 内联），无外部依赖，可直接在浏览器中打开。

核心功能

01
Squarified Treemap 布局算法
基于经典 Squarify 算法，按项目权重分配色块面积。权重由数据丰富度自动计算——数字数量与量级、描述文字长度等因素综合评分，经 score^1.6 指数放大形成最终权重。数据越丰富的项目占据越大面积。
02
冻结行树结构（Frozen Row Tree）
初始化时执行一次 Squarify 确定递归树形行结构（哪些 tile 在同一行、切割方向横或竖）。后续鼠标交互时只在行内按权重比例重新分配尺寸，tile 不会跨行重排。保证动画平滑无跳动，布局计算复杂度从 O(n log n) 降至 O(n)。
03
Canvas 文本预测量（Pretext）
借鉴 Pretext 库思路，初始化时用 Canvas measureText() 预计算所有文本在不同字号下的精确像素宽高，缓存为 TextMetricsCache。后续布局中根据 tile 实际尺寸与文本实际宽高做精确匹配——找到能放下的最大字号、判断是否需要换行、计算剩余空间决定显示哪些内容层级。整个过程为纯数学运算，零 DOM 查询。
预计算项：项目名 × 7档 | 数字 × 7档 | Badge × 7档 | 副标题 × 7档 | 描述 × 7档 决策逻辑：可用宽度 → 选最大字号 → 计算行数 → 累计已用高度 → 剩余高度决定下一层内容
04
光标感应交互（Gaussian Decay）
鼠标在 treemap 区域移动时，靠近光标的 tile 膨胀放大，远处收缩。采用高斯衰减函数 exp(-d²/r² × 2.5) 计算影响权重，最近 tile 放大至 3.5×，最远缩小至 0.65×。权重变化和位置均使用 lerp 平滑插值，帧率稳定流畅。
05
SDG 官方色彩系统
使用联合国 SDG 官方 17 色的淡色调（pastel tint）作为 tile 背景，17 个目标各有独立色系。鼠标悬浮时通过 HSL 插值动态提亮 + 阴影提升。导航栏数字 hover 时显示对应 SDG 鲜艳官方色。
06
内容自适应完整显示
所有 tile 始终显示 SDG 序号 + 名称（如 01. 无贫穷）和项目名。基于预测量文本尺寸，精确选择能容纳的最大字号，渐进显示数字、副标题、描述。无数字的 tile 描述文字吸底显示。无论 tile 多小，鼠标经过膨胀后内容自然完整展开。
07
导航栏形式设计
17 个 SDG 序号以 01 | 02 | ... | 17 排列，竖线高度为数字的两倍（28px），底部对齐。鼠标经过数字时对应 tile 高亮、其余变灰。纯展示用途，不可点击。
08
前言毛玻璃浮层
鼠标悬浮在标题文字上时，全屏毛玻璃遮罩层（backdrop-filter: blur(24px)）淡入，前言文字居中两端对齐显示。仅标题文字触发，年份切换器和导航栏不触发。鼠标离开标题立即消失。
09
年份 / 语言切换
标题右侧固定 2024 2023 年份切换，当前年份加粗，另一个可点击跳转。右上角固定 中 / EN 语言切换。四个页面交叉链接，年份切换保持语言不变，语言切换保持年份不变。
10
移动端探索模式（Bottom Sheet）
移动端（≤768px）采用全新交互范式：保留 Treemap 全貌作为视觉地图，点击色块后该色块高亮、其余变暗，底部滑出 Bottom Sheet 详情面板展示完整项目信息（SDG 标签、项目名、数据数字、副标题、完整描述）。面板内左右滑动可翻页浏览相邻项目，treemap 色块高亮联动。下滑面板或点击空白区域关闭。SDG 导航栏数字在移动端可点击，打开该 SDG 下第一个项目的详情面板。移动端禁用 Gaussian 光标膨胀效果，改为精准点击选中。
11
PC 端触屏兼容
PC 端（>768px）保留完整的鼠标悬浮膨胀交互，行为完全不变。触屏平板等大屏设备仍使用 Gaussian 探索模式。
12
入场动画
tile 以 opacity: 0 + scale(0.92) 初始状态，按顺序 stagger 淡入（每个延迟 30ms）。入场完成后切换为 JS 控制模式，移除 CSS transition 避免冲突。
13
性能优化
CSS contain: layout style paint 容器隔离；will-change: transform 合成层提升；元素引用创建时缓存避免每帧 querySelector；可见性状态 diff 检查跳过无变化 DOM 写入；动画循环收敛后自动停止（delta < 0.01）。

技术栈

纯原生 HTML5 + CSS3 + Vanilla JavaScript，无框架依赖
字体：Google Fonts（Inter + Noto Sans SC）
布局：Squarified Treemap + Frozen Row Tree（自研实现）
文本测量：Canvas 2D measureText() API（Pretext 思路）
动画：requestAnimationFrame + lerp 插值

浏览器兼容

Chrome / Edge / Safari / Firefox 最新版本。移动端 Safari / Chrome 触屏交互。需要支持 backdrop-filter（前言毛玻璃效果）。

使用方式

直接在浏览器中打开 sdg-treemap.html 即可查看 2024 中文版。通过标题旁的年份按钮和右上角的语言按钮切换不同版本。
